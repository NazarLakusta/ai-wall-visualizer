Контакти магазину

Клієнти бачать телефон і адресу в редакторі.
Консультант отримує заявки в окремий Telegram-чат (не в діалог клієнта з ботом).

Створіть групу → додайте бота → візьміть ID через @RawDataBot або @getidsbot

Якщо тестуєте самі — заявка змішається з чатом бота. Краще використовуйте групу вище.

Кожен магазин — свій бот зі своєю назвою в @BotFather. Після зміни токена перезапустіть docker compose up -d bot.

Якщо тест не приходить — відкрийте вашого бота в Telegram і натисніть /start, потім повторіть.
